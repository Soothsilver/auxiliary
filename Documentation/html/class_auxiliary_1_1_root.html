static void Auxiliary.Root.Init ( Game  game,
SpriteBatch  spriteBatch,
GraphicsDeviceManager  graphics,
Resolution  defaultResolution = null,
bool  fullscreenMode = false 
)
static

Binds this Root class to the graphics device from the game, and also sets the resolution and fullscreen/windowed mode. You must call Root.Init() before using any other Auxiliary functions.

Parameters
gameThe primary Game class containg the SpriteBatch and GraphicsDeviceManager.
spriteBatchThe spriteBatch Auxiliary will use to draw. Auxiliary will assume all calls to its method (that could potentially draw something) are inside spriteBatch.Begin() and spriteBatch.End() calls.
graphicsThe primary GraphicsDeviceManager from the Game class.
defaultResolutionThe resolution to set the game to, or null to leave unchanged.
fullscreenModeWhether to set the fullscreen mode or windowed mode.
